## 代码实现

```html
<div class="container" id="container">
    <!--拖动区域-->
    <section class="drag-area" id="drag-area"></section>
    <!--拖动按钮-->
    <div class="drag-tip" id="drag-tip"></div>
  </div>
```

```css
    .container {
      position: absolute;
      top: 0;
      left: 0;
      right: 0;
      bottom: 0;
      width: 100%;
      height: 100%;
      background-color: rgba(5, 9, 38, .8);
    }

    .drag-area {
      width: 300px;
      /*设置拖拽盒子的最小值，当拖拽的值小于这个值，将不再因拖拽而继续变小*/
      min-width: 100px;
      height: 100%;
      background-color: rgba(55, 165, 165, 0.3);
      position: relative;
    }

    .drag-tip {
      height: 100%;
      width: 6px;
      position: absolute;
      top: 0;
      left: 294px;
      background-color: transparent;
    }

    /*鼠标移入显示可拖动样式*/
    .drag-tip:hover {
      cursor: w-resize;
    }
```
