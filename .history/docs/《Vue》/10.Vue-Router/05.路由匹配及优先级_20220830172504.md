## 匹配规则

> 匹配404： /:notFound(.*)/*

```js
const routes = [
  // /:orderId -> 仅匹配数字
  { path: '/:orderId(\\d+)' },
  // /:productName -> 匹配其他任何内容
  { path: '/:productName' },
]
```

## 优先级

> 路由定义得越具体，定义得越早，优先级就越高。
