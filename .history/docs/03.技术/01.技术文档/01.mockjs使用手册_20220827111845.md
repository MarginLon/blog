## 安装

> yarn add mockjs --save-dev

## 语法规范

> 语法规范包括两部分：
>
> 1. 数据模板定义规范（Data Template Definition，DTD）
> 2. 数据占位符定义规范（Data Placeholder Definition，DPD）

### 数据模板定义规范 DTD

```javascript
// 属性名   name
// 生成规则 rule
// 属性值   value
'name|rule': value
```
