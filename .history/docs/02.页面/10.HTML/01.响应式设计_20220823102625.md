## container

````css
/*  */
.container {
  width: 100%;
}

/* max-width */
.container {
  max-width: 80%;
  margin: 0 auto;
}

/* media query */
.container {
  width: 800px;
}
@media (max-width:900px) {
  .container {
    width: 600px;
  }
}
@media (max-width:700px) {
  .container {
    width: 500px;
  }
}

````

## layout

```css
/* flex: 宽度过窄会出现异常 */
.container {
  display: flex;
}
/* 屏幕过窄会影响阅读，设置换行*/
.container {
  display: flex;
  flex: wrap;
} 
.container p {
  flex: 250px;
}

/* grid */
.container {
  display: grid;
}
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}
/* 屏幕过窄会影响阅读，设置换行*/
.container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
}
/* 或采取grid + media query 手动控制列数*/ 
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

@media (max-width: 900px) {
  .container {
    display: grid;
    grid-template-columns: 1fr 1fr;
  } 
}

@media (max-width: 600px) {
  .container {
    display: grid;
    grid-template-columns: 1fr;
  } 
}
```
