## table 样式

### table 的宽度设置无效

```css
.table {
  table-layout: fixed;
}
```

### 表格边框合并

```css
table {
  border-collapse: collapse;
}
```

### 完整 table 标签

```html
<!-- 完整版table -->
<table>
  <thead>
    <tr>
      <td></td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td></td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td></td>
    </tr>
  </tfoot>
</table>
```
